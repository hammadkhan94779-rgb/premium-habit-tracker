# 🌟 Nexus Habit - Premium Habit Tracker

A modern, AI-powered habit tracking application with gamification, analytics, and focus tools.

![Nexus Habit Preview](https://via.placeholder.com/800x400?text=Nexus+Habit+Preview)

## ✨ Features

### 🎯 Core Features
- **Habit Tracking** - Create, complete, and monitor daily habits
- **Streak System** - Track consecutive completions with animated flames
- **XP & Leveling** - Earn XP for consistency and level up your character
- **AI Coach** - Personalized insights based on your progress

### 📊 Analytics
- Interactive charts showing weekly/monthly trends
- 90-day consistency heatmap (GitHub-style)
- Success rate analysis and predictions
- Smart productivity insights

### 🎮 Gamification
- Unlockable achievements and badges
- Milestone celebrations with confetti
- Level up system (Beginner → Master)
- Daily challenges and goals

### ⏱️ Focus Tools
- Pomodoro timer with customizable sessions
- Ambient sound options (Rain, Lofi, Nature)
- Focus session tracking
- Break reminders

### 🎨 Premium UI
- Glassmorphism design with floating elements
- Smooth animations with Framer Motion
- Fully responsive (mobile, tablet, desktop)
- Dark mode optimized

### 💾 Data & Privacy
- Local storage for complete data privacy
- Export/Import backup functionality
- No account required
- PWA ready for installable app
